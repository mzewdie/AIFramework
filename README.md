# AIFramework

A lightweight Python framework for building AI applications.

Goals

This project demonstrates how modern AI applications can be built
using reusable software engineering principles.

Current features:

- Conversation management
- Prompt builder
- Configuration management
- Provider factory
- Gemini client
- AI engine
- Unit and integration tests

Planned features:

- Chatbot facade
- OpenAI provider
- Claude provider
- Conversation summarization
- Streaming
